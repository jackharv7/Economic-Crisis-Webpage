Support Vector Regression Model





Support Vector Regression Model is a machine learning technique used to identify patterns in historical data and present what the model has learned. SVR is also considered to be a classification algorithm. In this algorithm the data is shifted back to the number of days the user would like to predict in order to lag the data, this ensures that the calculation does not take in data that has not occurred yet. This model will then be trained on 80% of the data allocated from Yahoo Finance, then this model will be tested on the other 20% of data in order to predict future prices. This calculation will only be as accurate as the number of days the user would like to predict. The model will not be able to accuratley predict the next 30-50 days of prices due to other factors that influence stock prices that the machine will not be able to interpret. The accuracy of the model is much better when asked to predict the next 1-5 days becasue it will be able to rely on the most recent prices in order to make appropriate predictions.
